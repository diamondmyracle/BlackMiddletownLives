# Black Middletown Lives
**Final Project — Middletown, Connecticut and the Underground Railroad**

A multi-page website examining Middletown, CT's role in the Underground Railroad as a center of Black community life, abolitionist activism, and the movement of freedom seekers northward toward Canada.

## Site Structure

| File | Description |
|---|---|
| `index.html` | Homepage with interactive Leaflet.js map of the east-coast Underground Railroad route |
| `south.html` | Southern Origins — context on slavery, resistance, and the start of the Railroad |
| `middletown.html` | Middletown, CT — the way station, the Beman family, AME Zion churches, fugitive slaves |
| `canada.html` | Freedom in Canada — Black communities in Ontario and the meaning of liberty |
| `css/style.css` | Shared period-appropriate stylesheet |
| `js/map.js` | Leaflet map initialisation with route polyline and three clickable markers |

## Primary Sources

- Horatio T. Strother, *The Underground Railroad in Connecticut*, Chapter 11 ("Middletown, a Way Station")
- Cheryl Janifer LaRoche, "Family, Church, Community: Pillars of the Black Underground Railroad Movement," *Free Black Communities and the Underground Railroad* (2014)
- Cheryl Janifer LaRoche, "Appendix: Ministers Chart," *Free Black Communities and the Underground Railroad* (2014)
- Donohue and Bayers, "Fortresses of Faith, Agents of Change: AME and AME Zion Churches in Connecticut," in *African American Connecticut Explored*

## Research Questions Addressed

1. How did Black churches, families, and abolitionists in Middletown contribute to freedom-seeking and antislavery activism? *(Middletown page — Beman family, AME Zion church)*
2. In what ways did Middletown serve as a "way station" in Underground Railroad networks? *(Middletown page — Strother chapter)*
3. What was the general path freedom seekers took once passing through Middletown? *(Middletown page — route section; homepage map)*

## How to View

Open `index.html` in any modern web browser. An internet connection is required for the Leaflet map tiles and library to load.
